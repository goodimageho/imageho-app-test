# 퍼스널컬러 셀프테스터 - 1단계 웹사이트 설계서

## 📌 프로젝트 개요

### 목적
퍼스널컬러 4계절 진단 중 **1단계: 신체색 웜/쿨 체크**를 온라인에서 셀프로 진행할 수 있는 인터랙티브 웹사이트 개발

### 핵심 기능
1. **음성 안내** - 전문 컨설턴트의 안내 스크립트를 음성으로 제공
2. **퍼스널컬러 개념 설명** - 웜톤/쿨톤의 기본 개념 안내
3. **인터랙티브 체크표** - 사용자가 직접 클릭하여 체크할 수 있는 진단표
4. **결과 집계** - 웜/쿨 체크 개수 자동 카운트 및 결과 표시

## 🔊 음성 스크립트 (TTS 또는 녹음용)

### 1. 인사 및 소개 (intro.mp3)
```
안녕하세요. 퍼스널컬러 셀프테스터에 오신 것을 환영합니다.
자신에게 잘 어울리는 컬러를 찾는 퍼스널컬러 진단을 시작해볼까요?
```

### 2. 퍼스널컬러 개념 설명 (concept.mp3)
```
퍼스널컬러란 개인의 고유 신체색인 머리카락 색, 눈동자 색, 피부색, 
볼의 혈색, 입술색 등과 자연스럽게 조화를 이루는 컬러를 말합니다.

퍼스널컬러는 크게 웜톤과 쿨톤 두 가지로 분류합니다.
봄과 가을 계절은 각 컬러에 노란색이 들어가 있어 따뜻함이 느껴져서 웜톤이라고 합니다.
여름과 겨울 계절은 각 컬러에 파란색이 들어가 있어 시원함이 느껴져서 쿨톤이라고 합니다.
```

### 3. 1단계 시작 안내 (stage1-intro.mp3)
```
1단계 신체색 웜쿨 체크를 시작하겠습니다.
웜쿨 체크 시트에서 피부색, 입술색, 볼 혈색, 눈동자색, 머리카락색이 
웜쿨 어느 쪽에 가까운지 가볍게 체크해보세요.
```

### 4. 립앤치크 가이드 (lip-cheek-guide.mp3)
```
먼저 입술색과 볼의 혈색을 확인해보세요.
화면의 립앤치크 컬러 중 가장 가까워 보이는 색상을 선택해주세요.
웜톤의 1번부터 4번, 쿨톤의 1번부터 4번 중에서 선택하시면 됩니다.
```

### 5. 피부색 가이드 (skin-guide.mp3)
```
다음은 피부색입니다.
귀 앞쪽이나 목 부분의 피부색을 기준으로 확인해보세요.
웜톤과 쿨톤의 스킨 컬러 중 가장 가까워 보이는 색상을 선택해주세요.
```

### 6. 헤어앤아이 가이드 (hair-eye-guide.mp3)
```
마지막으로 눈동자색과 머리카락색을 확인해보세요.
자연광 아래에서 눈동자와 모발의 색상을 확인하시고,
가장 가까워 보이는 색상을 선택해주세요.
염색을 하셨다면 염색한 모발 기준으로 체크하시면 됩니다.
```

    coolCount,
